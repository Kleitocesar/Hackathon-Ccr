API´s desenvolvidas em Java com Spring Boot utlizando a estrutura padrão do Spring Boot.
As API´s são conectadas ao APP, mas foram pensadas que podem ser acopladas a outras interfaces,
O banco de dados utilizado foi o SQL Server, mas por utilizar JPA pode ser  qualquer banco de dados já que a modelagem é criada de forma automática, portanto, isso  depende do momento da hospedagem.
